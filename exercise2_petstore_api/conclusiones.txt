Conclusiones - Exercise 2
-------------------------
- Se automatizaron las operaciones solicitadas usando Cypress y cy.request para interacción con la API.
- Se utilizaron aserciones sobre códigos HTTP y sobre el cuerpo de respuesta para validar las operaciones.
